# Favicons à créer

Pour compléter l'optimisation SEO, vous devez créer les fichiers favicons suivants :

## Fichiers requis

1. **favicon.svg** - Favicon vectoriel (recommandé)
2. **favicon-32x32.png** - 32x32 pixels
3. **favicon-16x16.png** - 16x16 pixels
4. **apple-touch-icon.png** - 180x180 pixels (pour iOS)
5. **favicon-192x192.png** - 192x192 pixels (pour PWA)
6. **favicon-512x512.png** - 512x512 pixels (pour PWA)

## Outils recommandés

- [Favicon Generator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)

## Design recommandé

Utilisez le logo Venio avec la couleur orange (#FF6600) sur fond transparent ou sombre.

## Note

Les références à ces fichiers sont déjà configurées dans `index.html` et `manifest.json`.
