# Sauvegarde du Design Actuel

Ce dossier contient une sauvegarde complète du design neumorphism pur (avant le mélange avec le style cyber).

## Contenu

- `current-design/` : Tous les fichiers CSS du design original

## Pour revenir au design précédent

1. Copier les fichiers depuis `current-design/` vers `src/` et `src/components/`
2. Ou restaurer depuis Git si vous avez commité avant les changements

## Design actuel

Le design actuel est un **hybride** qui mélange :
- **Neumorphism** : Ombres douces, effets de relief
- **Style Cyber** : Clip-path polygon, effets néon, animations shimmer
- **Couleur** : Jaune fluorescent (#FFFF00)
